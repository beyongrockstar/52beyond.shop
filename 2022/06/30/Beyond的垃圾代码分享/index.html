C++入门

要开始使用 C++，你需要做两件事：
用于编写 C++ 代码的文本编辑器，例如记事本
编译器，如 GCC，将 C++ 代码翻译成计算机可以理解的语言。首先安装 IDE，IDE（集成开发环境）用于编辑和编译代码。
流行的 IDE 包括 Code::Blocks、Eclipse 和 Visual Studio。这些都是免费的，它们可用于编辑和调试 C++ 代码。（注意：基于 Web 的 IDE 也可以工作，但功能有限）我们将在教程中使用Code::Blocks，我们认为这是一个很好的起点。您可以在http://www.codeblocks.org/找到最新版本的 Codeblocks 。下载mingw-setup.exe文件，它将安装带有编译器的文本编辑器。

C++ 语法

让我们分解以下代码以更好地理解它:
例子:

include

using namespace std;

int main() {
cout << “Hello World!”;
return 0;
}

示例说明
第 1 行： #include 是一个 头文件库，让我们可以处理输入和输出对象，例如 cout（第 5 行中使用的）。头文件向 C++ 程序添加功能
第 2 行： using namespace std 意味着我们可以使用标准库中的对象和变量的名称
如果你不了解如何工作，请不要#include 担心using namespace std。只要把它想象成（几乎）总是出现在你的程序中的东西
第 3 行：空行。C++ 忽略空格。但是我们使用它来使代码更具可读性。
第 4 行：另一个总是出现在 C++ 程序中的东西是int main(). 这称为 函数。大括号内的任何代码{}都将被执行。
第 5 行：（ cout发音为“see-out”）是 与插入运算符( ) 一起用于输出/打印文本的对象。在我们的示例中，它将输出“Hello World”。<<

注意：每个 C++ 语句都以分号结尾

注意：的主体int main()也可以写成：
int main () { cout << “Hello World! “; return 0; }

请记住：编译器会忽略空格。但是，多行使代码更具可读性。
第 6 行： return 0结束 main 函数。
第 7 行：不要忘记添加右大括号}以实际结束 main 函数。

省略命名空间

您可能会看到一些在没有标准命名空间库的情况下运行的 C++ 程序。该using namespace std行可以省略并替换为std关键字，后跟::某些对象的运算符：
例子

include

int main() {
std::cout << “Hello World!”;
return 0;
}
